# cjcomp.py

This is my Python composing package for 2025 and 2026.

## Variable conventions

- p: a pitch in p-space represented in MIDI note numbers [12, 13, 14, ..., 127]
- pc: a pitch-class in pc-space [0, 1, 2, ..., 11]
- pchord: an ordered set of pitches in a chord arranged from low to high
- pseg: an ordered set pf pitches in p-space


## Other files in this collection

### enum_

Files that start with enum_ will create lists and tables that could be used for reference, in an Obsidian note, for example.

### gen_

Files that start with gen_ are meant to generate musical material that will be loaded into a Dorico file.

### my_

Files that start with my_ are things that will probably end up as functions or methods.
